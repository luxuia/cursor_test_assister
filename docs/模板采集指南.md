# 模板采集指南

## 📸 方法一：使用模板采集工具（推荐）

### 步骤

1. **打开模板采集工具**
   - 点击主窗口的"模板采集"按钮
   - 或运行程序后选择"工具" -> "模板采集"

2. **捕获游戏画面**
   - 点击"捕获游戏画面"按钮
   - 或点击"加载图片文件"从本地图片提取

3. **选择模板区域**
   - 在图片上拖拽鼠标选择要提取的区域
   - 红色框显示选中的区域
   - 右侧预览窗口显示裁剪后的模板

4. **设置模板信息**
   - 输入模板名称（如：hero_001）
   - 选择分类（Heroes/Equipment/Skills/UI）
   - 点击"保存模板"

5. **模板保存位置**
   - Heroes模板 → `Templates/Heroes/`
   - Equipment模板 → `Templates/Equipment/`
   - Skills模板 → `Templates/Skills/`
   - UI模板 → `Templates/UI/`

## 🎮 方法二：从游戏截图手动提取

### 准备工作

1. **游戏截图**
   - 在游戏中按截图键（通常是F12或PrintScreen）
   - 或使用Windows截图工具（Win+Shift+S）

2. **图片编辑工具**
   - 使用画图、Photoshop、GIMP等工具
   - 或使用在线图片编辑器

### 提取步骤

1. **打开截图**
   - 用图片编辑工具打开游戏截图

2. **裁剪模板**
   - 选择要提取的区域（如英雄头像）
   - 裁剪为合适的大小（建议32x32到128x128像素）

3. **保存模板**
   - 保存为PNG格式（保持透明度）
   - 文件名使用有意义的名称（如：hero_001.png）
   - 保存到对应的分类目录

## 📥 方法三：从网络资源获取（需注意版权）

### 注意事项

⚠️ **版权警告**：使用网络图片时请注意：
- 确保有使用权限
- 尊重原作者版权
- 仅用于个人学习和研究

### 推荐资源

1. **游戏官方资源**
   - 游戏官网的素材库
   - 官方API提供的资源

2. **开源资源**
   - GitHub上的游戏资源项目
   - 开源游戏素材库

3. **社区资源**
   - 游戏社区分享的素材
   - 玩家制作的资源包

### 下载和使用步骤

1. **下载图片**
   - 从上述资源下载需要的图片
   - 确保图片清晰，背景简单

2. **预处理**
   - 调整图片大小
   - 去除背景（如需要）
   - 统一格式为PNG

3. **导入模板**
   - 将处理好的图片复制到对应的模板目录
   - 使用模板采集工具验证模板质量

## 🔧 模板优化建议

### 模板质量要求

1. **清晰度**
   - 图片清晰，无模糊
   - 分辨率适中（32x32到128x128）

2. **背景**
   - 尽量使用透明背景
   - 或使用与游戏UI一致的背景色

3. **尺寸**
   - 保持原始尺寸比例
   - 不要过度缩放

4. **命名规范**
   - 使用有意义的名称
   - 建议格式：`类型_编号.png`
   - 例如：`hero_001.png`, `equipment_sword_001.png`

### 模板分类建议

#### Heroes（英雄）
- 英雄头像图标
- 英雄状态图标
- 英雄技能图标

#### Equipment（装备）
- 装备图标
- 装备品质标识
- 装备属性图标

#### Skills（技能）
- 技能图标
- 技能冷却状态
- 技能等级标识

#### UI（界面元素）
- 血条样式
- 按钮图标
- 状态指示器

## 📋 批量采集技巧

### 从同一张截图提取多个模板

1. **捕获完整游戏画面**
2. **依次选择不同区域**
3. **为每个区域设置不同的模板名称**
4. **逐个保存**

### 自动化采集（高级）

可以使用脚本批量处理：
1. 批量截图
2. 自动裁剪
3. 批量重命名
4. 自动分类

## 🐛 常见问题

### Q: 模板识别不准确怎么办？

A: 
1. 检查模板清晰度
2. 确保模板与游戏画面中的元素一致
3. 调整识别区域配置
4. 尝试多尺度匹配

### Q: 模板太大/太小？

A:
- 使用图片编辑工具调整大小
- 保持原始比例
- 建议尺寸：32x32到128x128像素

### Q: 如何更新模板？

A:
- 直接替换模板文件
- 或删除旧模板，添加新模板
- 程序会自动重新加载

### Q: 模板数量有限制吗？

A:
- 理论上没有限制
- 但模板过多可能影响识别性能
- 建议每个分类不超过100个模板

## 💡 最佳实践

1. **统一风格**
   - 保持模板风格一致
   - 使用相同的背景处理方式

2. **定期更新**
   - 游戏更新后及时更新模板
   - 保持模板与游戏版本同步

3. **版本管理**
   - 为模板添加版本号
   - 备份重要模板

4. **测试验证**
   - 采集后立即测试识别效果
   - 根据结果调整模板
